# CommentVerse AI 🎙️🌌

A futuristic AI-powered sports commentary web app where live match events are transformed into commentary styles from different universes.

## Features
- **Live Match Event Feed**: Simulated live events.
- **Multiverse Commentary Modes**: 9 unique styles (Anime, Shakespeare, Tamil Mass Hero, etc.).
- **AI Commentary Generator**: Powered by Google Gemini 1.5 Flash.
- **Voice Playback**: Listen to the commentary in real-time.
- **Dynamic UI**: Theme colors and animations change based on the selected universe.
- **Crowd Energy Meter**: Visualizing the stadium vibe.
- **Shareable Cards**: Generate and download social-media-ready commentary cards.

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion (Animations)
- Google Gemini API
- Web Speech API (Voice)
- html2canvas (Sharing)

## Setup
1. Clone the repo.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   VITE_GEMINI_API_KEY=your_api_key_here
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
